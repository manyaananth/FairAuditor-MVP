# ⚖️ FairAuditor MVP

An enterprise-grade AI bias detection dashboard built for the Solution Challenge 2026.

### Overview
Algorithmic bias in enterprise AI systems leads to unfair outcomes. FairAuditor allows data science teams to mathematically audit historical datasets for demographic parity before deploying machine learning models.

### Key Features
* **Real-Time Auditing:** Calculates Demographic Parity Difference using the `fairlearn` engine.
* **Interactive Visualization:** Dynamic plotting of disparate impact across demographic groups.
* **AI Mitigation Reports:** Leverages the Google Gemini AI model to instantly translate complex mathematical disparities into actionable executive summaries.

### Tech Stack
* **Frontend:** Streamlit
* **Data Processing:** Pandas, Fairlearn
* **Visuals:** Plotly
* **AI Integration:** Google Gemini API
